# 🚀 Ragroog
A minimal, hands-on implementation of a Retrieval-Augmented Generation (RAG) pipeline. This repository serves as my personal development, training, and deep-dive version of the well-known mini-rag course created by Abo Bakr Soliman.

The goal of this project is to demystify the core mechanics of RAG by building the foundational pieces from scratch, moving beyond simple API wrappers to understand data ingestion, chunking, embedding generation, vector storage, and semantic retrieval orchestration.
